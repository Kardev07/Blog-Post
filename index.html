Which programming language did I learn first?

To be honest, I learnt the Python programming language. It was really simple and easy to understand when it came to other languages.
And if you are a beginner, I would recommend learning a simple and easy language to understand. Otherwise, you might be stuck at the hands of a language which can be really confusing for you.

Which programming language do I learn first?

Actually there are a lot of programming languages to learn as a beginner. But you should learn the language which can be used to make the stuff you are most interested in.
The python programming language has many applications in which yours might be one. But there are some other languages which you might be interested in.
Some of the easy, simple languages to learn as a beginner are C# .NET, Html and CSS and Python
